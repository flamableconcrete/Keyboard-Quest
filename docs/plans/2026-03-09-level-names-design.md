# Level Names Design Document
**Date:** 2026-03-09
**Topic:** Level Names Redesign

## Overview
The game currently has level names that inconsistently use the player's available typing letters or don't fit the overall vibe. This redesign updates the names across all 5 worlds to be cohesive with a "Classic Heroic Fantasy" theme, reminiscent of classic tabletop RPG campaigns.
